CSS Background

The background property in CSS allows you to control the background of any elements (what paints underneath the content in that element). It is a shorthand property, which means that it allows you to write what would be multiple CSS Properties in one.

background is made up of eight other properties:

  • background-image
  • background-position
  • background-size
  • background-repeat

You can use any combination of these properties that you like, in almost any order.
