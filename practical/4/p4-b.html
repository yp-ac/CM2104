Terminologies in Web

HTML
Hyper Text Markup Language, Used to define the content of the webpage.
CSS
Cascading Style Sheets, defines the style for the website.
JavaSript
JavaSript(JS) is a scripting language, which you can add in website to make it functional. it can interact with DOM, and makes changes in it.
DOM
Document Object Model, it is the model of webpage, it has all the html elements present in it.
URL
Uniform Resource Locator, it is used to locate a specific document on web.
Responsive Design
A responsive design is what adapts to the screen size of the user's device, weather it be a mobile device, a tablet or a desktop.
Visual Hierarchy
visual hierarchy refers to the arrangement of design elements on a page so that the most important elements appear to have more emphasis.
SaaS
Software as a Service, In simple terms this means any service that provides a software platform, delivered from or via the cloud. for example, google docs, youtube
Server
A web server is a computer that server the website to the client/user.
Web Browser
An application software for accssing the web
Website
A website is a collection of web pages deployed under a domain
